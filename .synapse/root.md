# 🧠 Ariadne Engine: Root

## Project: NatohReMKT (WhatsApp Remarketing System)
**Description:** A web-based WhatsApp remarketing tool using `whatsapp-web.js` to manage labels, groups, and automated flows.

## Architecture
- **Frontend:** Vite + React (Aesthetics focused)
- **Backend:** Node.js + Express
- **WhatsApp Engine:** `whatsapp-web.js`
- **Real-time:** Socket.io
- **Database:** SQLite (Drizzle ORM for speed)

- [ ] Vite
